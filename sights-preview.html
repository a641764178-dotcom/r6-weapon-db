📸 游戏内军械库 1.0x 瞄具总览

布局说明：上中下三行排列。第 1 行 Red Dot 红点（方壳圆点/圆筒T字/抬高型）；第 2 行 Holographic 全息（EOTech 风/圆筒环/紧凑 Razor/极简 MH1）；第 3 行 Reflex 反射（塑料框三角/金属框三角/复古低矮）。点击图片可看大图。
Red Dot A · 红点 NATO
方形外壳 + 圆点准星。北约系干员的默认红点，最常见外观。
Red Dot B · 红点 Russian
宽镜体 + T 字准星。俄系/斯派兹纳兹风格。
Red Dot C · 红点 M4S Y5S3
抬高型 + 圆点准星。Y5S3 引入，镜身更高。
Holographic A · 全息 NATO
EOTech 风格方框 + 圆环内带点准星。
Holographic B · 全息 Russian
圆筒形外壳 + 圆环十字准星。俄系风格。
Holographic C · 全息 Razor
紧凑低矮款，镜身比 NATO/Russian 明显更小。
Holographic D · 全息 MH1 Y5S3
极简窄框，准星收束得非常小。Y5S3 引入。
Reflex A · 反射 NATO
塑料外壳 + 倒三角准星。常见反射镜。
Reflex B · 反射 Russian
金属外壳 + 倒三角准星，镜身更厚重。
Reflex C · 反射 Vintage
复古低矮款 + 倒三角准星。镜身最扁平。
